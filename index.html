구분 추천 운영 확인할 것
전주 → 완도항 트럭은 자전거와 짐, 일부 인원 이동. 남은 인원은 기차/버스 조합 또는 추가 차량으로 이동. 더블캡 탑승 정원, 운전자 피로, 야간 이동 시 교대 가능 여부
완도 → 제주 한일고속페리 완도-제주 노선 우선 확인. 새벽/오전 편을 쓰면 1일차 라이딩 시간이 넉넉합니다. 여객 8명, 트럭 1대, 자전거 8대 선적 가능 여부
제주 내 이동 3일 동안 전원 자전거 이동. 트럭은 항구, 숙소 짐 이동, 비상 상황에만 활용. 트럭 운전자 1명 고정 시 라이딩 참여 여부를 사전에 결정
제주 → 완도 3일차 제주항 도착 후 자전거 선적, 저녁 배편으로 복귀. 마감 시간 기준 최소 2시간 여유 확보. 기상, 출항 지연, 차량 선적 마감, 전주까지 야간 운전 가능 여부
한일고속페리 안내 기준으로 차량 선적은 출항 1시간 30분 전부터 가능하고 마감은 출항 30분 전입니다. 자전거 선적은 1대 3,000원으로 안내되어 있으며 선박별 제한이 있을 수 있습니다.

Preparation

팀 준비물과 운영 규칙

자전거 필수

  • 헬멧, 전조등, 후미등, 장갑, 우비
  • 예비 튜브 개인당 1개 이상, CO2 또는 펌프 2개 이상
  • 체인링크, 멀티툴, 타이어 레버, 보조배터리
  • 물통 2개 또는 하이드레이션팩

팀 운영

  • 선두 1명, 후미 1명 지정
  • 20~25km마다 전체 regroup
  • 강풍 구간은 2열 주행 금지, 도로 가장자리 낙차 주의
  • 식당 예약은 8명 기준으로 점심 1곳, 저녁 1곳만 고정

트럭 활용

  • 완도항과 제주항 승하선, 짐 이동에 사용
  • 숙소 체크인 전 짐 보관, 공구와 응급키트 보관
  • 라이딩 중 상시 동행 차량으로 쓰지 않음
  • 비상 회수 시 탑승 정원 때문에 순차 회수 필요

예약 순서

  • 1. 출발일 확정
  • 2. 배편 여객, 트럭, 자전거 선적 가능 여부 확인
  • 3. 숙소 2곳 예약, 자전거 보관 확인
  • 4. 트럭에 못 타는 인원 교통편 확정
